# Cat
Sirve para mostrar y concatenar archivos de texto
```bash
cat nota.txt
------------------------------------------------------------------------
Hola mundo!
```
## Opciones adicionales
- **-n** Muestra el número de cada línea
- **-E** Pone un $ al final de cada línea
- **-s** Combina varias líneas vacías en una sola
