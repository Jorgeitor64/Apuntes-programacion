# Cd
Permite navegar por el sistema de archivos cambiando el directorio de trabajo
```bash
cd [directory]
```
## Comandos útiles
- **cd** Para ir al directorio raíz
- **cd -** Para ir al directorio anterior
- **cd ..** Ir al directorio padre del actual
```bash
cd ../../dir3
```
