# Echo
Muestra texto o variables por pantalla
```
echo "Hola mundo"
------------------------------------------------------------------------
Hola mundo
```
Tiene diversas opciones
## Variables
```
name = Jorge
echo "Hola, $name"
------------------------------------------------------------------------
Hola, Jorge
```
## Múltiples líneas
Para imprimir múltiples líneas en una sola línea echo se usa \n
```
name = Jorge
echo "Hola, $name"
------------------------------------------------------------------------
Hola, Jorge
```
