# Network
## Definición
Una **network** es un conjunto de dispositivos interconectados que pueden comunicarse entre sí para compartir información, recursos y servicios siguiendo una serie de reglas conocidas como protocolos.
## Elementos básicos
- **Nodos:** dispositivos dentro de la red (ordenadores, servidores, móviles, IoT).
- **Medio de transmisión:** canal por el que viajan los datos (cable, fibra óptica, Wi-Fi).
- **Protocolos:** reglas que permiten la comunicación (ej: TCP/IP).
## Tipos de redes
- **LAN (Local Area Network):** red local (casa, instituto).
- **WAN (Wide Area Network):** red de gran escala (ej: Internet).
- **MAN (Metropolitan Area Network):** cubre una ciudad.
