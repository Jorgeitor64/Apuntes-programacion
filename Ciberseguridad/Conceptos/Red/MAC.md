# Dirección MAC (Media Access Control)
La **dirección MAC** es un identificador único de 48 bits asignado por el fabricante a una tarjeta de red (Network Interface Controller/NIC). Se considera el identificador físico de un dispositivo.
## Formato y Estructura
Se representa en formato hexadecimal, dividido es seis grupos de dos dígitos (ej. `00:1A:2B:3C:4D:5E`)
